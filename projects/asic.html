This project was made in collaboration with Zach Baldwin.

Github repo for the project

Overall Design

The design comprises of:
  • a control interface
  • 2 oscillators
  • a modulator
  • an ADSR (WIP)
  • digital filter (WIP)
    • .

    Oscillator

    Overall, the project comprised of creating a direct digital synthesis (DDS) algorithm, with a maximum frequency of roughly 20kHz at a resolution of 1/3 Hz, enabling pretty much every musical note to be played. The algorithm follows a basic DDS routine, having a phase accumulator, and different algorithms for producing different waveforms, including triangle, saw, pulse, and sine. The sinewave is done using a quarte-sine look-up table (LUT), which is generated with MATLAB. The other are generated via bit-tricks with the phase accumulator.

    Modulator

    The modulator modulates oscillator one with modulator two. The modulation routines are:
    • Amplitude via a radix-2 sequential multiplier
    • Addition without carry-out
    • Subtration with carry-out
    • Addition with carry-out
    • Magic Something (see Verilog code)
    • Oscillator 1 only
    • Oscillator 2 only
    • Bitwise XOR
    • Bitwise AND


    Current Portions

    The current portion of the project is design and implementation of an ADSR as well as a variable low pass filter.
